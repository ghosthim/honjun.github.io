//新建文章
hexo n 文章名
//新建单页面
hexo n page 页面名
//重新生成
hexo g
//提交
hexo d
//生成+提交
hexo g -d
//备份
hexo b
//清空生成内容
hexo clean
//默认启动服务在4000端口
hexo s
//启动服务在5555端口
hexo s -p 5555
